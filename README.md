# Smart Factory Data Platform

Progetto di simulazione di una piattaforma dati per ambienti industriali,
ispirata a contesti di produzione manifatturiera.

## Obiettivo
Costruire una pipeline ETL per raccogliere, trasformare e analizzare dati
di produzione di fabbrica.

## Stack Tecnologico
- PostgreSQL
- Python
- SQL
- Git
- AWS (livello concettuale)

## Stato del progetto
Fase 0 – Setup iniziale del repository e dell’ambiente
